
## 03_pjt 반응형 웹 페이지 구현 

### 1. 프로젝트 개요본 

프로젝트는 금융 상품 비교 애플리케이션의 사용자 프로필 페이지를 HTML과 CSS, Bootstrap5를 활용하여 반응형으로 구현하는 것을 목표로 합니다. 
특히 뉴진스의 Hype Boy 컨셉을 차용하여 Y2K 레트로 감성과 하이틴 무드를 웹 UI에 녹여내었습니다.

### 2. 개발 환경 및 도구
언어: HTML5, CSS3 
프레임워크: Bootstrap 5.3.3 
폰트: 구글 폰트 Hahmlet (100~900)
도구: Visual Studio Code 

### 3. 핵심 요구사항 분석 및 구현 내용

#### (1) F301 프로필 및 투자 성향 출력  
프로필 사진: rounded-circle 클래스를 사용해 원형으로 출력하고, 6px의 두꺼운 그림자(box-shadow)를 적용하여 키치한 느낌을 강조했습니다.

자기소개: 사진 하단에 textarea와 Update 버튼을 배치하여 사용자가 직접 내용을 입력하고 수정할 수 있는 UI를 구성했습니다.

투자 성향: 프로필 사진 우측에 배치하였으며, 투자상품 선호도와 관심 종목은 체크박스(복수 선택), 투자 스타일은 라디오 버튼(단일 선택)으로 구현했습니다.

**(2) F302 회원 정보 출력**  

편집 제한 필드: 회원 번호와 ID는 readonly 속성을 부여하고 배경색을 bg-light로 설정하여 편집이 불가능한 필드임을 명시했습니다.

유효성 고려 입력 필드: Email은 올바른 형식을 유도하기 위해 input type="email"을 사용했고, 나이와 자산 및 연봉은 input type="number"를 사용하여 숫자만 입력 가능하도록 구현했습니다.

**(3) F303 네비게이션 바 구현**  

상단바: John Doe님의 프로필 페이지라는 타이틀을 담은 레트로 스타일의 상단바를 구현했습니다. 
페이지 이동: 기본 정보 수정과 포트폴리오 수정 페이지 사이를 자유롭게 이동할 수 있도록 탭 형태의 네비게이션 메뉴를 구성했습니다.

**(4) F304 포트폴리오 수정 페이지 구현**  

저축 성향: 도전형, 검소형, 알뜰형 중 하나만 선택할 수 있도록 라디오 버튼 그룹을 생성했습니다.
최애 은행: 텍스트 입력이 가능한 인풋 창과 업데이트 버튼을 결합하여 구성했습니다.

**(5) F305 반응형 디자인 구성**  
그리드 시스템: Bootstrap의 row와 col-md-6를 사용하여 가로 사이즈 768px 이상에서는 좌우 배치를 유지하고, 768px 미만에서는 요소들이 세로로 쌓이도록 설계했습니다.

스피너 제거: 숫자 입력창(type="number")에서 나타나는 기본 화살표 버튼을 CSS를 통해 숨김 처리하여 모바일 환경에서도 깔끔한 입력을 지원합니다.

### 4. CSS 전략: NewJeans 'Hype Boy' 컨셉

컬러 팔레트: 하이틴 감성의 파스텔 블루(#8bb8ff)와 톡톡 튀는 핑크(#ff9cee)를 주 색상으로 사용하여 청량한 무드를 연출했습니다.
UI 요소: 윈도우 팝업 느낌을 주기 위해 모든 박스에 두꺼운 검은색 테두리와 오프셋 그림자를 적용했습니다. 버튼은 마우스 클릭 시 실제로 눌리는 느낌이 나도록 transform 효과를 부여했습니다.
배경 디자인: Y2K 디지털 레트로 감성을 위해 연한 블루 톤의 모눈종이(Grid) 패턴을 그라데이션으로 배경에 적용했습니다.

### 5. 학습 내용 및 어려웠던 점

**학습 내용**: Bootstrap5의 유틸리티 클래스만으로 구현하기 힘든 세밀한 디자인 요소를 커스텀 CSS로 보완하는 방법을 익혔습니다. 
특히 복잡한 형태의 box-shadow와 input-group의 레이아웃 조절 능력을 키웠습니다.

**어려웠던 점**: 16진수 색상 코드 작성 시 오타로 인한 렌더링 오류가 발생했으나, 개발자 도구를 통해 CSS 문법 오류를 찾아 해결했습니다. 또한 숫자 입력창의 기본 스피너를 제거하면서 브라우저별(Chrome, Firefox) 호환성을 맞추는 과정에서 미디어 쿼리와 벤더 프리픽스의 중요성을 깨달았습니다.

### 6. 프로젝트 후기
단순한 정보 입력 페이지를 넘어서 특정 아티스트의 컨셉을 웹 디자인으로 녹여내는 과정이 매우 흥미로웠습니다. 기능 구현이라는 필수 목표를 달성하면서도 사용자에게 시각적인 즐거움을 줄 수 있는 UI/UX 설계의 중요성을 체감할 수 있었던 소중한 시간이었습니다
